# AI Advisor

PDFを解析してGeminiで要約とアドバイスを生成するシンプルなシングルページアプリです。管理画面でプロンプトとAPI設定を編集できます。

## 主な機能
- PDFドラッグ&ドロップアップロード、年齢/身長/体重入力
- Geminiによる要約・食事アドバイス・ビタミン/ミネラル指導・運動提案の生成
- 生成結果をPDFで出力
- 管理画面でプロンプト、Gemini APIキー、モデルを編集（ローカルストレージ保存）
- APIキー未設定時はサンプル結果を表示してUIを確認可能

## セットアップ
```bash
npm install
npm run dev
```
ブラウザで表示されたローカルURLへアクセスしてください。

## 使い方
1) 管理画面(`/admin`)でGeminiのAPIキーとモデル、プロンプトを入力して保存  
2) ホーム画面の「PDF解析を始める」から解析ページへ移動  
3) PDFをドロップ or 選択し、年齢/身長/体重を入力  
4) 「Geminiで解析する」で要約とアドバイスを生成  
5) 必要に応じて「PDFで出力」でレポートをダウンロード

## 注意点
- APIキーはブラウザのローカルストレージのみに保存されます。共有環境では扱いに注意してください。
- プロンプトもローカルに保持されます。チームで共通管理する場合は別途バックアップしてください。

## ビルド
```bash
npm run build
```
